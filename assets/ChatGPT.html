To use a modal in a page without causing a re-render when opening the modal, you typically want to conditionally render the modal component without directly affecting the rest of the page's components. Here's a basic approach:

import Modal from './Modal'; // Import your modal component

  const [isModalOpen, setIsModalOpen] = useState(false);

  const openModal = () => {
    setIsModalOpen(true);
  };

  const closeModal = () => {
    setIsModalOpen(false);
  };

  return (
    <div>
      <h1>My Page</h1>
      <button onClick={openModal}>Open Modal</button>
      {isModalOpen && <Modal onClose={closeModal} />}
    </div>
  );
};

export default MyPage;

In this example:

  • We have a state variable isModalOpen which controls whether the modal is open or closed.
  • When the "Open Modal" button is clicked, it sets isModalOpen to true, causing the modal to be rendered.
  • The modal component (<Modal />) is only rendered if isModalOpen is true. When isModalOpen becomes true, React will only mount the <Modal /> component onto the DOM at that point without re-rendering the rest of the page.
  • The modal component has an onClose prop (or similar) to handle closing the modal. This allows the modal to communicate with the parent component to change the isModalOpen state when the modal is closed.

This way, opening and closing the modal won't cause a re-render of the rest of the page. Only the modal component itself will mount and unmount when necessary.
